git init -> git remote -> git pull(레파지토리가 비워져 있으면 할 필요x) => git clone(레파지토리의 이름을 딴 파일이 생성된다.)

-> git add .(어떤파일의 변동사항을 저장할지 *파일선택*)

-> git commit -m "변동사항에 대한 상세한 설명"(변동사항 저장)

-> git push origin master(깃허브에 변동사항 업로드)
